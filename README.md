### Projets

[**urbex-detector**](https://github.com/ludwigyoda)(bientot public sans géoloc)
Pipeline qui prend 900 000 entrées brutes de spots d'urbex, les déduplique, les normalise, les géocode, et sort 50 000 adresses propres avec index spatial dans Postgres.

[**patient-hub**](https://github.com/ludwigyoda)(accès sous demande)
App React Native qui capte l'accéléromètre en arrière-plan et déclenche un rapport médical au format SOAP quand certains seuils sont franchis.

[**realtime-dashboard**](https://github.com/ludwigyoda)(accès sous demande)
Dashboard admin avec push WebSocket, requêtes pré-agrégées Postgres et cache LRU côté serveur. Plusieurs widgets se rafraîchissent à des fréquences indépendantes selon leur criticité.
